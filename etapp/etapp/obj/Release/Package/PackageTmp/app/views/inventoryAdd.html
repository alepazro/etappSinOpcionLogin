  • Create an excel spreadsheet with the information of the devices you want to upload.
  • When asked, please select the type of device you are uploading. IMPORTANT: This upload utility can handle one type of device at a time. For instance, if you wanted to upload GenX CDMA and GenX HSPA, please upload one type first, and then the other, so you would need to execute this process 2 times.

  • The excel must have the following columns with these exact names:

    SerialNumber IMEI CARRIER/SIMTYPE/MVNO SIM/ESN/MEID MSISDN/MDN

    *Notes

    • SerialNumber: Mandatory
    • IMEI: Optional. Leave blank if you want, but include the column in the excell nontheless.
    • CARRIER/SIMTYPE/MVNO: Mandatory. Enter here an identifier of the type of sim. Examples: ATT, VERIZON, RACO, KORE... You can come up with your own sim identifier.
    • SIM/ESN/MEID: Mandatory. This is the sim card number or the ESN for CDMA devices.
    • MSISDN/MDN: Mandatory. This is the 'phone number' of the device. Please enter JUST NUMBERS (ex. 5553332222). Do NOT include dashes or spaces or parenthesis.
    You can download an excel template here:
    Download Template (Excel 97-2003)
    Download Template (Excel 2007 and above)
  • Copy / Paste the data from the excel spreadsheet into the box below.
  • Click Import Data

Please select the device type. Please upload one type of device at a time.
